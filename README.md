# Extractor de modelos de Rails
MDA Reverse enginnering tool that produces 2 model diagrams (UML class diagram like) from Ruby on Rails applications, for helping to take decisions in development.

## Technologies  
- Eclipse 
- Xtext
- Epsilon ETL
- Sirius
